## Hướng dẫn cài đặt & chạy project

### 1. Cài đặt dependencies cho Client

```bash
cd Client
npm install
```

### 2. Cài đặt dependencies cho Backend

```bash
cd Backend
npm install
```
### 3. Cài đặt dependencies cho Admin

```bash
cd Admin
npm install
```
### 3. Chạy project

**Mở 3 terminal:**

- **Terminal 1:** Chạy backend
  ```bash
  cd Backend
  npm start
  ```

- **Terminal 2:** Chạy Client
  ```bash
  cd Client
  npm start
  ```
  
  - **Terminal 1:** Chạy Admin
  ```bash
  cd Admin
  npm start
  ```
